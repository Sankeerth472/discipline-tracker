# Discipline Tracker

A lightweight browser app for planning 1-month goals, linking daily tasks to them, and tracking points-based progress.

## Features

- Add long-term monthly goals with point targets
- Log daily tasks and map each task to a long-term goal
- Mark tasks complete to earn points
- Track day-wise scores and recent history
- See goal completion, streak, monthly activity, and completion rate
- Data persists in browser `localStorage`

## Run

Open `index.html` in a browser.

If you want a local server instead of opening the file directly:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

- Your data is saved locally in the browser on your machine.
